# AI Resume Analyzer & Career Assistant

An AI-powered web application that analyzes resumes, extracts skills,
scores job suitability, and provides career guidance using NLP.
Built with MERN stack and open-source AI tools.

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB
- AI/NLP: Python, spaCy, TF-IDF
